# smctl delete-broker

## Overview
`smctl delete-broker` deletes a set of brokers registered in the Service Manager instance.

## Usage
```bash
smctl delete-broker [name] <name2 <name3> ... <nameN>> [flags]
```

## Aliases
```bash
delete-broker, db 
```
